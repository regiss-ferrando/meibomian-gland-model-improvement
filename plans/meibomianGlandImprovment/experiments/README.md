# Experiments

This directory contains experiment runs organized by strategy.

## Subfolders
- `01_baseline/`: baseline segmentation and score model experiments.
- `02_data_augmentation/`: experiments focused on augmentation and data-centric improvements.
- `03_transfer_learning/`: transfer learning and self-supervised pretraining experiments.
- `04_architecture_change/`: architecture upgrades, multi-task models, and ensemble experiments.

Each folder should include a short experiment summary, config file, and result log.
