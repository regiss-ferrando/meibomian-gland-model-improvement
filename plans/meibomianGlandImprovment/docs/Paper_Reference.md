# MGD-1k Dataset Paper Reference

## Citation
**Automated quantification of meibomian gland dropout in infrared meibography using deep learning**

- **Published in:** Contact Lens and Anterior Eye
- **DOI:** 10.1016/j.clae.2022.101611
- **ScienceDirect link:** https://www.sciencedirect.com/science/article/pii/S1542012422000519

## How to Access
1. **Through institutional access:**

2. **Preprint versions:**
   - Check ResearchGate: https://www.researchgate.net/

   - Check arXiv: https://arxiv.org/

3. **Direct from authors:**
   - Contact paper authors for a preprint copy
   - MGD-1k GitHub: https://github.com/MGD1K/MGD1k

4. **Dataset documentation:**
   - The MGD-1k dataset GitHub repository contains supplementary materials
   - Dataset page: https://mgd1k.github.io/

## Key Information for Project
- Dataset size: 1,000 infrared meibomian gland images
- Annotations: Segmentation masks and graded meiboscore (0-3)
- Application: Meibomian gland dropout detection and quantification
- Task: Segmentation + classification/regression for dry eye assessment
